**Bienvenue sur le référentiel de test**

Ce site contient du contenu fake destiné à tester le système de synchronisation upstream entre plusieurs référentiels.

## Contenu de test

Ce référentiel contient :

- **3 fiches** de bonnes pratiques exemples
- **2 personas** de test
- **3 termes** de lexique

## Architecture multi-sites

Ce repo (`gen-referentiel-core`) sert de source pour synchroniser le code entre plusieurs sites :

- **RWP** - Référentiel WordPress
- **RWEB** - Référentiel Web
- **REIPRO** - Référentiel Intégration Progiciels

Seul le contenu (`src/content/`) diffère entre les sites. Le code est partagé via upstream sync.

## À propos

Ce contenu est volontairement minimaliste pour :

1. Tester que le build fonctionne
2. Valider les liens entre fiches, personas et lexique
3. Démontrer l'architecture de synchronisation
